# ai-random-fill-script

**[JP]** Adobe Illustratorで選択したオブジェクトの「塗り」をランダムな色に変更するスクリプト。  
**[EN]** An Adobe Illustrator script that applies random fill colors to selected objects.
![https://raw.githubusercontent.com/K-inoue3719/ai-random-fill-script/refs/heads/main/demo.png](https://raw.githubusercontent.com/K-inoue3719/ai-random-fill-script/refs/heads/main/demo.png)

## 特徴 / Features

* **Simple:** 選択してスクリプトを実行するだけの簡単操作。
* **Time-saving:** 大量のオブジェクトへの着色作業を一瞬で完了させます。
* **Random:** 実行するたびに異なる色が適用されます。

## 動作環境 / Requirement

* Adobe Illustrator (CC / 2020 / 2024 etc.)
    * *動作確認済み: Adobe Illustrator 2024 (Windows/Mac)*

## 使い方 / Usage

### 1. インストール (Installation)

1.  このリポジトリから `ai-random-fill-script.jsx` をダウンロードします。
2.  スクリプトファイルを任意の場所に保存します。

### 2. 実行 (Run)

1.  Adobe Illustratorを開き、色を変更したいオブジェクトを選択します。
2.  メニューバーから `ファイル(File)` > `スクリプト(Scripts)` > `その他のスクリプト(Other Script...)` を選択します。
3.  保存した `ai-random-fill-script.jsx` を選択して開きます。
4.  選択したオブジェクトの色がランダムに変更されます。

> **Tips:** よく使う場合は、Illustratorのスクリプトフォルダにファイルを入れると、メニューに常時表示されるようになります。
>
> * **Windows:** `C:\Program Files\Adobe\Adobe Illustrator [Version]\Presets\ja_JP\スクリプト`
> * **Mac:** `/Applications/Adobe Illustrator [Version]/Presets.localized/ja_JP/Scripts`

## 注意事項 / Note

* グループ化されたオブジェクトに対しては個別のパスごとに色が変わります。
* 実行前にデータのバックアップを推奨します。
